# Data pipeline analysis

The included file "assets.json" contains descriptions of tables generated in an ETL process, with known flaws.
Some descriptions and documentation is intentionally missing, and figuring it out by context is part of your task.

Please provide brief responses to each task below. Include any notes, diagrams (encouraged), and/or code as supplemental information.

Perform the following tasks:

1. Explain the steps taken to calculate "trfm_school_board_reports_long".
2. What are the columns in that table, and where do they come from?
3. What problems or innefficiencies are there in that calculation? How could it be improved?
4. Complete the JSON for the incomplete asset `trfm_scores_by_domain`.
5. Describe the process you would take to verify and update the `schema_simple` properties if you were provided all tables.
